What does an OS do?

Ans) The main task for an Operating System is:

  • Virtualization : Virtualization of CPU and Memory form User Programs.
  • Concurrency : To run user programs in parallel/simultaneously.
  • Persistence : File System.
  • Security : Process Isolation.

These are the 4 fundamental issues or concerns that an OS must consider.

Only the first 3 will be covered in class and Security will be done through a project.
